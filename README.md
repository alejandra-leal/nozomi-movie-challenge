# Nozomi Movie App

Nozomi comes from the word "hope" in japanese, doesn't really have anything to do with movies, but we should never lose it.

## Prerequisites 

- [NodeJS](https://nodejs.org/en/download/ 'NodeJS') installed

## Quickstart

### Step 1: Install dependencies

`npm install`

### Step 2: Start app

`npm start`

Enjoy!
